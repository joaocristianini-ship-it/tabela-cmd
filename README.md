# tabela-cmd
# 🚀Guia de Sobrevivencia: Domínio do Terminal (CLI)

Este repositório foi criado como parte da disciplina de **Sistema Operacionais**, utilizando o simulador [Terminal Master](http://andersonrmgomes.github.io/terminal-master/) para praticar comandos essenciais de Windows e Linux.

---

## 📖 Introdução
A interface de linha de comando (CLI) é uma ferramenta poderosa que permite interagir com o núcleo do sistema operacional. Dominar esses comandos é fundamental para automação de tarefas, administração de servidores e desenvolvimento de software.

--

## 🖥️ Comandos aprendidos

Abaixo, apresento a lista dos principais comandos praticados durante o treinamento gamificado:

|Comando |Função |Sistemas | Exemplo de Uso |
| :--- | :--- | :--- | :--- |
| `pwd` | `exibe o diretorio atual (caminho)` | `Linux` | `pwd` |
| `cd` | `altera o diretorio de trabalho` | `ambos` | `cd /home/usuario/documento` |
| `ls` | `lista arquivos e diretórios ` | `Linux ` | ` ls -la` |
| `dir` | `lista arquivos e diretórios ` | `Windows ` | `dir ` |
| `mkdir` | `cria um novo diretório ` | `ambos ` | ` mkdir nova_pasta` |
| `touch` | `cria um arquivo vazio ou atualiza a data ` | `Linux ` | ` touch arquivo.txt` |
| `type` | `exibe o conteúdo de um arquivo ` | `Windows ` | ` type arquivo.txt` |
| `rm` | `remove arquivos ou diretórios ` | `Linux ` | ` rm arquivo.txt` |
 `cp ` | `copia arquivos ou diretórios ` | ` ambos` | `cp origem.txt destino.txt ` |
  `mv ` | `move ou renomeia arquivos/diretórios ` | `ambos ` | `mv arquivo.txt nova_pasta/ ` |
| `cat` | `exibe o conteudo de arquivos` | `linux` | `cat arquivo.txt` |
| `more` | `exibe conteúdo página por página` | `ambos` | `more arquivo.txt` |
| `less` | `visualiza arquivos de forma interativa` | `linux` | `less arquivo.txt` |
| `echo` | `imprime texto no terminal` | `ambos` | `echo "Olá"` |
| `clear` | `limpa o terminal` | `linux` | `clear` |
| `cls` | `limpa o terminal` | `windows` | `cls` |
| `rmdir` | `remove diretorios vazios` | `ambos` | `rmdir pasta` |
| `del` | `remove arquivos` | `windows` | `del arquivo.txt` |
| `copy` | `copia arquivos` | `windows` | `copy a.txt b.txt` |
| `move` | `move arquivos` | `windows` | `move a.txt pasta\` |
| `rename` | `renomeia arquivos` | `windows` | `rename a.txt b.txt` |
| `find` | `busca arquivos ou texto` | `ambos` | `find "texto" arquivo.txt` |
| `grep` | `busca texto dentro de arquivos` | `Linux` | `grep "erro" log.txt` |
| `chmod` | `altera permissões de arquivos` | `Linux` | `chmod 755 script.sh` |
| `chown` | `altera dono do arquivo` | `Linux` | `chown usuario arquivo.txt` |
| `df` | `mostra uso de disco` | `Linux` | `df -h` |
| `du` | `mostra tamanho de diretórios` | `Linux` | `du -sh pasta` |
| `history` | `mostra histórico de comandos` | `Linux` | `history` | 
| `head` | `mostra as primeiras linhas de um arquivo` | `Linux` | `head arquivo.txt` |
| `tail` | `mostra as últimas linhas de um arquivo` | `Linux` | `tail arquivo.txt` |
| `ping` | `testa conexão com um host` | `ambos` | `ping google.com` |
| `ipconfig` | `mostra configurações de rede` | `Windows` | `ipconfig` |
| `ifconfig` | `mostra configurações de rede` | `Linux` | `ifconfig` |
| `whoami` | `mostra o usuário atual` | `ambos` | `whoami` |
| `exit` | `encerra o terminal` | `ambos` | `exit` |
